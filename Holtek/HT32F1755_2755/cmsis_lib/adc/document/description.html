ADC

Overview

  • A 12-bit ADC is an approximation method for Analog to Digital Converter. There are totally 10 multiplexed channels with 8 external channels which can measure signals from analog input and 2 internal sources. If the input voltage is beyond the thresholds, the feature of Analog Watchdog will detect the outranged signal. An interrupt will then be generated to warn that the input voltage is higher or lower then the thresholds. There are three conversion modes to translate a data from analog to digital. Various channels of the A/D conversion can be operated in one shot, continuous and discontinuous mode. A left-aligned or right-aligned 16-bit data register is provided to store the data after conversion.

API Reference

    ADC_DeInit

    Deinitializes the ADCx peripheral registers to their default reset values.
    ADC_Init

    Initializes the ADC peripheral according to the specified parameters in the ADC_InitStruct.
    ADC_Reset

    Reset ADC.
    ADC_IntConfig

    Enables or Disables the specified ADC interrupts.
    ADC_GetIntStatus

    Checks whether the specified ADC interrupt has occurred.
    ADC_GetFlagStatus

    Checks whether the specified ADC flag has been set.
    ADC_ClearIntPendingBit

    Clears the ADCx interrupt pending bits.
    ADC_ClearFlag

    Clears the ADC pending flags.
    ADC_ExtTrigRegularConvConfig

    Configures the ADCx external trigger for regular channels conversion.
    ADC_ExtTrigHPConvConfig

    Configures the ADCx external trigger for high priority channels conversion.
    ADC_RegularGroupConfig

    Configures cyclic conversion mode and length of list queue for regular group.
    ADC_HPGroupConfig

    Configures cyclic conversion mode and length of list queue for high priority group.
    ADC_GetHPConversionData

    Returns the result of ADC high priority channel conversion.
    ADC_GetConversionData

    Returns the result of ADC regular channel conversion.
    ADC_RegularChannelConfig

    Configures the corresponding rank in the sequencer and the sample time for the regular channel of the selected ADC.
    ADC_HPChannelConfig

    Configures the corresponding rank in the sequencer and the sample time for the high-priority channel of the selected ADC.
    ADC_SoftwareStartConvCmd

    Enables or Disables software start of the regular channel conversion of the selected ADC .
    ADC_SoftwareStartHPConvCmd

    Enables or Disables software start of the high priority channel conversion of the selected ADC .
    ADC_AWDConfig

    Enables or Disables Lower/Upper threshold warning of the analog watchdog on single/all channels.
    ADC_AWDSingleChannelConfig

    Configures the analog watchdog that guards single channel.
    ADC_AWDThresholdsConfig

    Configures the high and low thresholds of the analog watchdog.
