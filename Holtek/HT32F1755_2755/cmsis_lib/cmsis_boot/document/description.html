CMSIS_BOOT

Overview

  • This component is used to boot the HT32F125x device. It provides definition of the peripherals for the actual device, startup code and the complete (device specific) Interrupt Vector Table.

    API Reference

    __attribute__

    The minimal vector table for a Cortex M3. Note that the proper constructs must be placed on this to ensure that it ends up at physical address 0x00000000.
    Default_Reset_Handler

    This is the code that gets called when the processor first starts execution following a reset event. Only the absolutely necessary set is performed, after which the application supplied main() routine is called.
    Default_Handler

    Provide weak aliases for each Exception handler to the Default_Handler. As they are weak aliases, any function with the same name will override this definition.
    SystemInit

    Setup the microcontroller system. Initializes the system clocks and the embedded Flash.
    SystemCoreClockUpdate

    Update SystemCoreClock.
